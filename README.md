|<p>1. Crear un repositorio en GitHub con el nombre “Practica08 – Laberinto VR”</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.004.png)</p><p></p><p>Crear la GVR Camera Rig</p><p></p><p>Durante este paso, crearemos la cámara VR incluyendo el GvrEditorEmulator en la escena y configurando la cámara.</p><p></p><p>1.Agregue el objeto prefab GvrEditorEmulator a la escena.</p><p>2.Convierta el objeto Main Camera en un elemento hijo del objeto GvrEditorEmulator.</p><p>3.Restablece el componente Transformar del objeto Main Camera</p><p>4.Mueva el objeto GvrEditorEmulator a una ubicación conveniente para el desarrollo, por ejemplo, Posición: 0, 3, 35 y Rotación: 0, 180, 0.</p><p>5.Ingrese al modo de juego y use Alt + Mouse y Ctrl + Mouse para rotar e inclinar el ángulo de visión de la cámara.</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.005.png)</p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.006.png)</p><p></p><p>**Preparando la escena para la interacción**</p><p></p><p>Durante este paso, prepararemos la escena para la interacción configurando el puntero, el emisor de rayos y el sistema de eventos, y luego probaremos el sistema de punto de referencia incluido (waypoint).</p><p></p><p>1. Agregue el objeto prefab GvrReticlePointer a la escena como elemento secundario del objeto del Main Camera.</p><p>2. Aumente el valor de Distancia máxima de retícula para el GvrReticlePointer del valor predeterminado de 10 a 20.</p><p>3. Agregue el script GvrPointerPhysicsRaycaster como componente en el objeto Main Camera.</p><p>4. Agregue el objeto prefab GvrEventSystem a la escena.</p><p>5. Ingrese al modo de juego y navegue por la escena haciendo clic en los puntos de referencia.</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.007.png)</p><p></p><p></p><p></p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.008.png)</p><p></p><p>**Hacer que los objetos del juego sean interactivos**</p><p></p><p>Durante este paso haremos que la Moneda, la Llave y la Puerta sean interactivas añadiéndoles componentes de disparadores y eventos.</p><p></p><p>1. Localiza y selecciona el objeto Coin en la jerarquía.</p><p>2. Verifique que tenga un componente Collider.</p><p>3. Agregue el script Coin proporcionado como componente.</p><p>4. Agregue un dispardor de evento (Event Trigger) como componente.</p><p>5. Agregue el evento PointerClick al componente Event Trigger.</p><p>6. Asigne el componente del script Coin al campo de objeto del evento Pointer Click.</p><p>7. Asigne el método Coin.OnCoinClicked () como la función para el evento Pointer Click.</p><p>8. Ingrese al modo de juego, haga clic en la moneda y verifique que el mensaje 'Coin.OnCoinClicked ()' esté impreso en la ventana de la consola.</p><p>9. Repita el mismo proceso para el objeto del juego Key pero use el script Key y el método Key.OnKeyClicked ().</p><p>10. Repita el mismo proceso para el primer objeto principal del juego de Puerta, pero use el script de Puerta y el método Door.OnDoorClicked ().</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.009.png)</p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.010.png)</p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.011.png)</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.012.png)</p><p></p><p></p><p>**Hacer la interfaz de usuario interactiva**</p><p></p><p>1. Localiza y selecciona el objeto del juego SignPost en la jerarquía.</p><p>2. Elimine el componente Graphic Raycaster que se agrega automáticamente al crear un nuevo objeto.</p><p>3. Agregue el script GvrPointerGraphicRaycaster como componente.</p><p>4. Agregue el script SignPost proporcionado como componente.</p><p>5. Agregue un Event Trigger como componente.</p><p>6. Agregue el evento PointerClick al componente Event Trigger.</p><p>7. Asigne el componente script SignPost al campo de objeto del evento Pointer Click.</p><p>8. Asigne el método SignPost.ResetScene () como la función para el evento Pointer Click.</p><p>9.Ingrese al modo de juego, haga clic en SignPost y verifique que el mensaje 'SignPost.ResetScene ()' esté impreso en la ventana de la consola</p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.013.png)</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.014.png)</p><p></p><p></p><p></p><p></p><p></p><p>**Programando el comportamiento de la moneda (coin)**</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.015.png)</p><p></p><p>**Programando el comportamiento de la llave (key)**</p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.016.png)</p><p></p><p></p><p></p><p>**Programando el comportamiento de la puerta (door)**</p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.017.png)</p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.018.png)</p><p></p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.019.png)</p><p></p><p>**Programando el comportamiento del SignPost**</p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.020.png)</p><p></p><p></p><p>![](Práctica%20de%20laboratorio%2008%20-%20Realidad%20Virtual-Laberinto-BryanMendez.021.png)</p><p></p>|
